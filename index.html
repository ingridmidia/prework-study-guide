HTML

  • HyperText Markup Language (HTML).
  • Used for content and to create the structure of a webpage.
  • HTML elements are usually composed of an opening and closing tag represented with angle brackets.
  • The head element contains information about the webpage, or metada.
  • The body element represents the visible content shown to the user.
  • HTML attribute defines additional information about an element.

CSS

  • Cascading Style Sheets (CSS).
  • Defines the colors, aesthetics, visual layout, and organization of the webpage.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The preferred way to style a webpage is by using an external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • Git commands always start with git.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript is a dynamic language. Variables are not directly associated with any particular value type.
  • JavaScript is a weakly typed language, which means it allows implicit type conversion when an operation involves mismatched types, instead of throwing type errors.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
